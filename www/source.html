Source Code

We encourage people to read Gnusto's source code, and to make suggestions. Before you dive in, you might well be interested in reading a commentary on the code (which is still under development, but covers all the most important parts).

You may read the source in either of the following ways:

  • By using the CVS web interface.
  • By checking out the source using anonymous CVS with the following commands (the password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -d :pserver:guest@mozdev.org:/cvs co gnusto
