Installation

Please read the release notes for v0.6.0 before installing it. Thanks. Gnusto is still under development, and the release notes tell you the pitfalls you should be aware of.

If you're running under Unix, you can now install gnusto without logging in as root. Just choose the "install to profile" option when prompted during the installation. If you wish to install to the main program directory, you must do so as root.

If you have an earlier version of gnusto installed, you must uninstall it (the release notes for the previous version have instructions for a manual uninstall) and delete the entire gnusto program directory (found inside Mozilla's "chrome" directory) before installing. If that's not possible, you could also try reinstalling the browser. This version includes an uninstaller, so this step will not be required in future versions.

To install gnusto v0.6.0 (the September 2003 release), click on the picture below:

Starting gnusto

Then either:

  • restart Mozilla Firebird (as an ordinary user) and choose "Gnusto" from the Tools menu.
  • or start Mozilla Firebird from the command line thus:
    MozillaFirebird -chrome chrome://gnusto/content
