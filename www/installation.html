Installation

Please read the release notes for v0.5.0 before installing it. Thanks. Gnusto is still under development, and the release notes tell you the pitfalls you should be aware of.

If you're running under Unix, you must be running Mozilla as root to install add-ons. (You're adding new software, after all.) The exception is if you installed Mozilla originally as a non-root user, in which case you should run it as that user.

The old null bytes problem has been fixed.

To install gnusto v0.5.0 (the July 2003 release), click on the picture below:

Starting gnusto

Then either:

  • restart Mozilla Firebird (as an ordinary user) and choose "Gnusto" from the Tools menu.
  • or start Mozilla Firebird from the command line thus:
    MozillaFirebird -chrome chrome://gnusto/content
