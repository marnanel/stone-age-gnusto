Gnusto is not even prealpha yet. Don't expect anything to work.

Gnusto is a Z-machine, written in ECMAScript (which some call JavaScript). It's primarily designed to be used with Mozilla, though we hope other browsers will eventually work with it too.

You can try out a recent version implemented as a web page, if you're running a Mozilla-based browser (others may not work yet). Or you could download the most recent version and play with it yourself.

You can find story files for the Z-Machine at the IF Archive; if you'd rather not just dive in, Baf's Guide is an excellent place to start. Gnusto isn't good enough yet to run arbitrary games, though, so here's a list of other interpreters.

At gnusto's freshmeat page, you can subscribe to new releases, or vote for the project.

The name "gnusto" comes from a spell in the Enchanter series. Its use in this project was suggested by Boluc Papuccuoglu in a thread about the project on rec.games.int-fiction.
